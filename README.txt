To get started

1. Set env variable
	% export GITHOME_MSU=....femtoscopy_workshop

2. cd into the commonutils/software directory, and enter
   % cmake .
	% make
	then cd intto the coral/software directory, and enter
	% cmake .
	% make

3. cd into the myrun directory and enter
   % cmake .
	% make gausstest
	% gausstest
	
You can change the correlation by editing gausstest.cc
or the parameters/wfparameters.dat or the parameters/coralpars.dat
